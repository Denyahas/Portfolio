Introduction

HTML (HyperText Markup Language) is the standard markup language used to create web pages and web applications. It is used to describe the structure of web documents using elements that browsers interpret to display content. HTML is the foundation of every web page, and all websites on the internet are built using it.

The main purposes of HTML:

  • Structuring the web page.
  • Formatting text and including multimedia content.
  • Organizing links for navigation between pages.

Main HTML Elements

  • Structural Elements

    An HTML document begins with !DOCTYPE html, which indicates that the latest version of HTML is being used. Next comes the root html element that contains all the other elements of the document.

    • head — contains metadata (information about the document, links to CSS and JS).
    • title — the title of the page, displayed on the browser tab.
    • body — the main part of the page, where all visible content is placed.

  • Text Elements

    HTML provides various elements for formatting text:

    • h1> - <h6 — headings of different levels (from the largest to the smallest).
    • p — paragraph of text.
    • strong and em — for bold and italic text, respectively.
    • ul and ol — unordered and ordered lists, respectively.
    • list — list item.

  • Hyperlinks

    One of the key features of HTML is the ability to create links between pages using the a element:

    • The href attribute specifies the URL to navigate to.
    • Example: a href=https://example.com">Go to the website</a

  • Images

    Images are embedded in a webpage using the img element:

    • The src attribute specifies the path to the image file.
    • The alt attribute contains a description of the image for accessibility purposes or in case the image cannot be loaded.

  • Tables

    HTML supports creating tables for displaying data:

    • table — root element of a table.
    • tr — table row.
    • td — table data cell.
    • th — table header cell.

    Example:

    table
  tr
    th>Name</th
    th>Age</th
  /tr
  tr
    td>John</td
    td>30</td
  /tr
/table

  • Forms

    HTML provides powerful tools for creating forms, allowing interaction with the user:

    • form — the container for the form.
    • input — various input fields (text, email, password, date).
    • button — button to submit the form.

    Example:

    form action="/submit" method="POST"
  label for="name">Name:</label
  input type="text" id="name" name="name"
  button type="submit">Submit</button
/form

Graphics and Multimedia

HTML supports embedded multimedia elements:

  • img for images.
  • audio for audio.
  • video for video.

These elements allow direct playback of media files on the web page without additional plugins.

HTML5 — The Evolution of HTML

HTML5 expanded the capabilities of previous HTML versions by introducing new semantic elements and APIs. It allowed developers to create more interactive web applications, improved multimedia support, and provided better support for mobile devices.

New semantic elements:

  • header, footer — for the header and footer of a page.
  • article, section — for content structuring.
  • aside — for content related to the main content.

HTML5 also introduced APIs for working with local storage (LocalStorage), geolocation, and graphics (Canvas).

Metadata and SEO

Metadata is information about a web page that is not displayed to users but is used by browsers and search engines:

  • meta charset="UTF-8" — specifies the document's character encoding.
  • meta name="description" content="Page description" — a description of the page for search engines.
  • meta name="keywords" content="keywords" — keywords for SEO.

Conclusion

HTML is the foundational language for creating web pages. It provides basic tools for structuring and formatting content, as well as modern capabilities through HTML5 for working with multimedia and interactive elements. Understanding the basics of HTML is essential for anyone looking to develop websites or web applications.
